# Developer Agent

Owns the Developer Repo: receives confirmed milestones, produces technical designs, implements, and reports back.

## Workflow

1. **Receipt** — accept incoming Milestone Confirmation File
2. **Technical design** — produce TDD, ADRs, API contracts, ERD (Q6)
3. **Validation** — run Gate 2 technical design self-check (Q7)
4. **Implementation planning** — brainstorm options, break into atomic pieces, build timeline (Q8)
5. **Implementation** — execute atomic changes with tests (Q9-Q10)
6. **Report** — promote decisions-log to ADRs, generate Implementation Report, archive
