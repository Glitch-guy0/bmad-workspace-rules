# Milestone Stress Test

Validates a milestone before sign-off.

## Checks

1. **Clarity** — is every requirement clear and unambiguous?
2. **Edge cases** — what happens at boundary conditions?
3. **Scope boundaries** — is everything in scope strictly necessary?
4. **Conflict detection** — do any requirements conflict?
5. **Dependency reveal** — are all dependencies identified?

## Outcome

- All checks pass → sign-off proceeds
- Open items found → resolve before sign-off
