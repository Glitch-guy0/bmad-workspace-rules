# 🧭 The Bigger Shift Happening

Frontend engineering is splitting into **5 major tracks** now:

| Track                       | Focus                        |
| --------------------------- | ---------------------------- |
| App Frontend                | dashboards/SaaS              |
| Creative Frontend           | 3D/graphics/motion           |
| AI Frontend                 | copilots/agents              |
| Realtime Frontend           | multiplayer/collab           |
| Browser Systems Engineering | WASM/WebGPU/native-like apps |

And honestly…
the boundary between frontend and systems engineering is disappearing fast 🚀
